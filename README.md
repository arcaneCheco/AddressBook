## Getting started
#### 1. Clone the repository
<pre>
git clone https://github.com/arcaneCheco/AddressBook.git ./addressBook
cd ./addressBook
</pre>
#### 2. Install Dependencies
<pre>
npm install
</pre>
#### 3. Launch servers
<pre>
npm run serve
</pre>
#### 4. Visit http://localhost:1234
#### 5. Run unit tests
<pre>
npm run test
</pre>
